1 Take a clean copy of [Whitelabel PWA](https://github.com/listingslab-software/whitelabel-pwa)

```bash
cd <your-project>
npm install @listingslab-software/pushtotalk@0.0.9
```

```javascript
<PushToTalk options={{
	api: `https://api.listingslab.com`,

```
